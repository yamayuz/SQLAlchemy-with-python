# python-practice-06 - try to SQLAlchemy with python + mysql on docker
## Setup & Run
dockerコンテナを起動
```
docker-compose up -d
```

dockerコンテナに入る
```
docker exec -it try-to-sqlalchemy /bin/bash
```

python実行
```
python main.py
```
